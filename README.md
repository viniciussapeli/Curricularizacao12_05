# Curricularizacao12_05
Repositório criado a fim de registrar a realização da atividade de curricularização da faculdade de sistemas de informação, UNIFEBE

Explicação do jogo a ser criado: O jogo a ser criado ira ser no estilo quiz, onde a criança terá uma série de perguntas a serem respondidas, conforme mais respostas corretas, aparecera uma tela dos resultados com uma mensagem positiva, se o resultado for negativo, também aparecera uma mensagem positiva mas com o intuito de motivar a melhorar na próxima vez que tentar responder.
